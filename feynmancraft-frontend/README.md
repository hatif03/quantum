# FeynmanCraft Frontend

Standalone interactive frontend for the Particle Physics Agent.

## Open

Open `index.html` in a browser. No build step is required.

## What it includes

- Animated particle-field hero and interactive Feynman diagram canvas
- Plain-language story for visitors who do not know Feynman diagrams
- Example process prompts based on the agent README
- Six-agent workflow section matching the ADK project architecture
- Example TikZ-Feynman output panel

The original `Particle-Physics-Agent-main` folder was only read for context and was not modified.
